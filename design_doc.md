# 网页设计文档：当前时间展示页面

## 设计思路
设计一个简洁、清晰、实时更新的网页，用于展示当前时间。采用极简风格，以时间显示为核心，背景使用渐变色增加视觉层次感。页面无需用户操作即可自动刷新，适合作为仪表盘或浏览器新标签页使用。

## 布局
页面采用 Flexbox 实现垂直水平居中布局，时间显示在页面中央，使用大号字体突出显示。日期显示在时间下方，字体稍小。整体布局响应式，适配移动端和桌面端。时间下方可选提供12/24小时制切换按钮。

## 样式
- 背景：从深蓝 (#1a1a2e) 到紫色 (#16213e) 的渐变背景
- 时间：白色字体，font-family: 'Segoe UI', sans-serif，字号 4rem，字重 300
- 日期：白色字体，透明度 0.8，字号 1.2rem
- 切换按钮：圆角半透明背景，悬停效果
- 每秒通过 JavaScript 的 setInterval 更新一次时间显示

## 交互
页面加载后自动开始显示并每秒更新时间。用户可点击按钮切换12小时制与24小时制显示。日期格式为“YYYY年MM月DD日 星期X”。整体设计注重可读性和沉浸感。
